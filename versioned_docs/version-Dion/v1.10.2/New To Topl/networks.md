# Networks

There are two Topl networks as well as custom networks and developer networks. 

Our Brambl API library accepts the following networkPrefixes `toplnet, valhalla, local, private" where Toplnet is the main network, Valhalla is the testing network, local and private designate developer networks running in your local environment.
