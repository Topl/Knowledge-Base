To prove a transaction run the following command:

<pre><code>
brambl-cli tx prove -w $PASSWORD --keyfile {props.keyfile} -i {props.txFile} -o {props.txFileProved} --walletdb {props.wallet}
</code></pre>

<p>This will prove the transaction in the file <code>{props.txFile}</code> and store the result in the file <code>{props.txFileProved}</code>. The right indexes to derive the keys are taken from the wallet database.</p>
