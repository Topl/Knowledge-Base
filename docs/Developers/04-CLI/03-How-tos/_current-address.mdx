To get the current address of the wallet run the following command:

<pre><code>
brambl-cli wallet current-address --walletdb {props.wallet} --from-fellowship {props.fellowship} --from-template {props.template} --from-interaction {props.interaction}
</code></pre>

If you have several interactions already in your wallet, you can just omit the `--from-interaction`
parameter and the CLI will use the last interaction of the template `default`.
